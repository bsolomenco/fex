Detailed Description

Fixed-width vertical bar with Copy, Compare, and Search buttons.

The bar only emits signals — it has no knowledge of the file panels or any business logic. MainWindow connects the signals to the appropriate handler slots.

The widget uses object name "centerBar"; its buttons use "centerBarButton" — both are styled in style.qss.
