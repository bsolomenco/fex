Detailed Description

Read-only table dialog for directory comparison results.

Opened by CompareController after DirectoryComparator has finished. The dialog is modal and sets WA_DeleteOnClose.

Table columns: Name | Status | Left Size | Right Size | Left Modified | Right Modified
