Detailed Description

Recursive file-search service that streams results via Qt signals.

Decoupled from any dialog or widget — callers connect to resultFound and finished to receive results incrementally as the search runs.

Pattern matching rules

  • If the pattern contains * or ?, it is matched with QDir::match() (standard glob, case-insensitive).
  • Otherwise a plain case-insensitive substring search is used.

Threading

search() runs synchronously on the caller's thread. QApplication::processEvents() is called every 50 results so the UI remains responsive without requiring a background thread.
