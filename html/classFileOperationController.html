Detailed Description

Mediates between UI copy requests and the FileOperations service.

Depends only on QWidget* (for dialog parenting) and the stateless FileOperations namespace — it never touches FilePanel directly.

Initiates a confirmed, progress-tracked copy operation.

Shows a confirmation dialog, checks for destination conflicts (with an overwrite prompt), then copies each item using FileOperations::copyPath(). Emits copyCompleted when done.
