Detailed Description

Flat text button used inside ToolBarWidget.

Inherits QPushButton with a flat appearance and a pointing-hand cursor. All visual styling is delegated to style.qss via the ToolTextButton type selector — there are no inline stylesheets in this class.

Each button carries a ButtonType enum value that identifies which menu it represents, allowing controllers to act on the type rather than the display text.
