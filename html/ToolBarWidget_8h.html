Detailed Description

Menu bar widget displayed at the top of the main window.

Hosts eight ToolTextButton items (Files, Mark, Commands, Net, Show, Configuration, Start, Help), each wired to a QMenu populated with the standard Total Commander menu entries.

Left-side buttons are left-aligned; "Help" is pushed to the right.
