Detailed Description

Single-line command input bar displayed below the file panels.

Mirrors the command bar found in Total Commander: a prompt label on the left shows the active directory, and a QLineEdit on the right accepts typed commands. Pressing Enter emits commandEntered and clears the input field.

The prompt is updated automatically by MainWindow whenever the active panel changes directory.
