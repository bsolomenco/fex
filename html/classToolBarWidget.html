Detailed Description

The top-level menu bar containing all application menus.

The widget uses the Qt object name "menuBar" so that style.qss can target it with the selector #menuBar { … }.

Menu contents are built by static factory functions defined in the corresponding .cpp file and assigned to each button via QPushButton::setMenu().
