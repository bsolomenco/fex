Detailed Description

Pure-UI dialog that displays pre-computed directory comparison results.

This class has no knowledge of the filesystem — it receives a QList<CompareEntry> (produced by DirectoryComparator) and renders it as a colour-coded table. All business logic lives in the service layer.

Row colour CompareEntry::Status
Amber LeftOnly
Blue RightOnly
Red Different
Green Equal
