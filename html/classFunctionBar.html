Detailed Description

Horizontal row of F1–F10 action buttons.

The widget uses the Qt object name "functionBar"; each individual button uses "functionButton" — both are styled in style.qss.

Currently the bar only emits signals. Connecting them to actual operations (View, Edit, Copy …) is done in MainWindow.
