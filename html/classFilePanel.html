Detailed Description

A file-browser pane backed by QFileSystemModel.

The panel displays the contents of one directory at a time. Navigating into a subdirectory is done by double-clicking a folder row. Drive buttons at the top jump to the root of each mounted volume.

The widget uses the Qt object name "filePanel" so that style.qss can target it with #filePanel { … }.
