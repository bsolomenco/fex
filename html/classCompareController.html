Detailed Description

Mediates between the UI compare request and the DirectoryComparator service.

Owns a DirectoryComparator instance. On compare(), it:

  1. Delegates computation to the comparator.
  2. Opens a CompareDialog pre-populated with the results.

MainWindow creates one instance and calls compare() when the CenterBar emits compareRequested.
