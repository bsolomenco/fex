Detailed Description

Path-prompt widget and shell command input line.

Layout: [ /home/user> ] [ ________________________________ ]

The widget uses the Qt object name "commandBar". Its prompt label uses "commandLabel" and the input field uses "commandInput" — all styled in style.qss.
