Detailed Description

Single-pane file browser widget used inside the dual-panel layout.

Each FilePanel is self-contained: it owns a drive selector bar, a path label, and a QTableView backed by a QFileSystemModel. Two instances of this widget (left and right) are placed side-by-side in MainWindow.
