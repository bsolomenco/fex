Detailed Description

Wires a FileSearcher and a SearchDialog, relays navigation to MainWindow.

Handles the Open/Closed concern for search: if the search implementation ever changes (e.g. threaded, indexed), only this controller and FileSearcher need updating — SearchDialog and MainWindow stay untouched.
